# Admin Service

## Overview
The Admin Service is the central governance layer of the Career Identity Verification & Recruitment Platform. It manages privileged users (recruiters, executives, admins), enforces platform policies, and maintains audit logs.

**Tech Stack:**
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB (via Mongoose)
- **Authentication:** RS-256 JWTs (Verification) & Internal Secrets (Inter-service)
